# Dollars to Cents

Convert dollars to cents app.

## Instructions

To start this app:

- Installing Dependencies:

```
npm install
```

- Compiles and hot-reloads for development:

```
npm run dev
```

- Compiles and minifies for production:

```
npm run build
```

- Run your unit tests:

```
npm run test
```

- Run linter:

```
npm run lint
```

- Run formatter:

```
npm run fmt
```
